== To setup Angular you should do following == 
1. Create package.json
   NPM init - run this command in project directory and type parameters you need
2. Download Angular framework
   npm install @angular/common @angular/core @angular/compiler @angular/for ms @angular/http @angular/platform-browser 
   @angular/platform-browser-dynamic @angular/router --save
2a. For further reinstallation: npm install
3. Start web server (if it's needed)
   npm install lite-server -g
4. Download Module loader
   npm install systemjs --save
5. Create root component (app.module.ts)
